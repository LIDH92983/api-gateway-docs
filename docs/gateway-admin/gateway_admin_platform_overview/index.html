平台简介

概述​

网关管理平台是配合自研网关打造的一款综合管理工具，专注于场景、资源和规则的高效管理。它支持规则的动态发布、资源的绑定与审核，以及实时的运行监控，为网关的使用者提供一站式服务。通过此平台，用户可以直观地进行操作，简化配置流程，确保网关的高效运行。

功能模块​

1. 场景管理​

  • 功能：
    • 场景是网关运行的逻辑分组，可以灵活创建、编辑和删除。
    • 不同场景之间互相独立，便于开发、测试、生产等环境的隔离管理。
  • 特点：
    • 场景与资源绑定关系明确，可随时调整。
    • 支持快速切换场景，动态加载规则。
    • TODO：后续将实现场景配置的自动备份与一键还原。

2. 资源管理​

  • 功能：
    • 对网关中的资源进行全面管理，例如服务接口、文件目录等。
    • 支持多场景共享资源，也可为资源设置独立的场景规则。
    • 资源状态实时更新，便于用户快速了解其使用情况。
  • 特点：
    • Redis数据推送：资源本身的数据修改后会通过Redis订阅服务推送至网关，实现无缝更新。
    • 动态配置：资源修改后无需重启即可生效。
    • 审核流程：资源与规则绑定需通过审核后方可启用，确保配置的安全性和正确性。
    • 批量管理：支持批量新增、批量绑定及批量解绑资源。
    • 批量导入：支持手动输入、Excel导入及粘贴批量新增资源，提高配置效率。

3. 规则管理​

  • 功能：
    • 平台支持丰富的规则管理功能，包括创建、编辑、发布和删除。
    • 提供限流、允许、拒绝、时区等多种规则类型，并支持自定义扩展。
    • 每条规则都有版本控制，便于回溯和修改。
  • 特点：
    • 模板化：内置规则模板，提高规则配置效率。
    • 多场景适配：同一规则可以应用于多个资源，实现灵活性和统一性。
    • 发布管理：规则发布分为草稿、测试和正式三个阶段，避免误操作导致的服务中断。
    • 批量绑定与解绑：支持批量操作，提升规则管理效率。

4. 审核功能​

  • 功能：
    • 资源与规则绑定或修改时，需经过审核流程后生效。
    • 审核过程支持多级审批，满足不同组织的需求。
  • 特点：
    • 审核历史记录完整，可随时追溯。
    • 未通过审核的修改将被退回到草稿状态，确保稳定性。
    • 通过审核的数据将通过Redis订阅服务推送至网关，实现无成本更新。
    • TODO：后续将增加审核自动化功能，进一步提升效率。

5. 监控大盘​

  • 功能：
    • 提供多维度的监控数据展示，包括：
      • 请求统计：实时监控每秒请求量、成功率和失败率。
      • 规则分析：统计规则命中次数，分析热点规则。
      • 资源监控：展示各资源的访问量及响应时间。
    • TODO: 后续可以考虑支持自定义告警规则，当指标异常时及时通知。
  • 特点：
    • 可视化：使用柱状图、折线图等直观展现数据。
    • 实时刷新：监控数据秒级更新，保障监控的准确性。
    • TODO：历史数据回溯与报表导出功能开发中。

6. 网关相关​

  • 功能：
    • 规则列表：实时展示网关当前正在运行的所有规则，包括规则类型、绑定资源及优先级等详细信息。
    • 请求记录：记录所有进入网关的请求，包含时间戳、来源 IP、目标资源及状态码等，便于问题排查。
    • 审核记录：展示所有审核列表的历史记录，包括审核状态、审核人及时间。
    • 同步记录：规则发布审核通过后，记录规则同步到网关引擎的详细信息，确保配置的落地。
  • 特点：
    • 支持多条件筛选与分页查看，提升记录查询效率。
    • 数据留存：请求及同步记录可自定义保留周期，减小存储压力。
    • 可追溯性：所有记录均可点击查看详情，为问题定位提供数据支撑。
    • TODO：后续将支持自动化数据归档，进一步提升存储管理效率。

平台亮点​

  1. 轻量化：平台核心功能均基于高效的缓存机制，查询性能优秀，操作响应迅速。
  2. 动态性：所有配置实时生效，无需重启服务。
  3. 可扩展性：平台架构模块化设计，方便新增功能。
  4. 全链路管理：从规则配置到发布、监控全流程覆盖，用户体验良好。

未来规划​

  1. 流量负载配置：针对每个资源，用户可以配置流量负载策略，在处理大规模请求时能够平衡负载，确保系统的稳定性。
  2. 针对资源可以：允许用户按需生成数据报表，支持导出。

网关管理平台通过模块化设计实现了场景、资源与规则的全方位管理，同时结合实时监控和数据记录功能，帮助用户全面掌控网关运行状态。未来，平台将继续拓展功能，为用户提供更加智能化的管理体验。
